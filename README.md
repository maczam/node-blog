node-blog
=======
---
node-blog是node编写一个超级简单的blog应用。 [在线demo](http://maczam.duapp.com/)

* 数据库使用当前最流行的mongodb
* 渲染引擎使用markdown
* 支持本地、vps和bae部署
* 支持更换主题

本地安装
---
1. node install
2. 修改settings.js配置文件local信息
2. 本地启动` node server.js local`

bae安装
---
1. 修改settings.js配置文件baidu信息
2. 部署
