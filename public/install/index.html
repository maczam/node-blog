node-blog简介

node-blog是开源node编写一个简单的blog应用。

  • 数据库使用当前最流行的mongodb
  • 渲染引擎使用markdown
  • 支持本地、vps和bae部署
  • 支持更换主题

安装

  • 支持vps
  • bae

vps安装

  • node install
  • 修改settings.js配置文件local信息
  • 本地启动 node server.js local

bae安装

  • 申请node运行单元
  • 申请mongodb
  • 修改settings.js配置文件baidu信息
  • 上传到bae并发布
